Iter와 enumerate함수

iter() 함수

iter()함수는 반복을 끝낼 값을 지정할 수 있다.

ls = list(range(10))

it = iter(ls)

for _ in range(len(ls) + 1):

print(next(it)) # ls의 값을 0 부터 9까지 출력 그 후에 StopIteration 예외 발생

enumerate() 함수

enumerate() 함수는 몇번째 반복문이지 확인이 가능하다.

튜플 형태로 (index, value)를 반환한다.

ls = list(range(10))

for item in enumerate(ls):

print(item, end=" ") # (0, 0) (1, 1) (2, 2) ...

또한 시작 인덱스 지정이 가능함.

ls = list(range(10))

for item in enumerate(ls, 1):

... # (1,0) (2,1) ...
